To get the kubeconfig for the cluster, run the following command:

- aws eks update-kubeconfig --name surv-2

To run the cluster:

- execute deploy and services of: video-processor, index and router
- make sure security group of nodes has open inbound rule
- check ip of index and router nodes
- put them in env file for webui
- build webui image and push it
- deploy and service webui
